## IoT-based-Hydroponics-Monitoring-System
Smart IoT-based hydroponics monitoring and automation system for real time environmental monitoring and control.
# Overview
This project aims to automate the monitoring and management of a conventional hydroponics setup by reducing manual intervention and providing real time environmental data. The long-term objective is to improve plant growth consistency through sensing, automation and cloud-based monitoring. 
# Hardware
- ESP32 Devkit Module with webcam
- 7 in 1 NPK Sensor
- DC Motor Pumps
- DHT11 (Temperature and Humidity Sensor)

# Current Features
- Real-time environmental monitoring
- Cloud connected data visualization
- Automated control of the primary water circulation pump and nutrient addition valves.
- Prototype successfully tested with Lettuce cultivation over 20 days of monitoring.
